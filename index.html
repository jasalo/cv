Trabajo

Lulo · App para iOS y Android
CTO & Cofundador

App para pedir mercado a domicilio desde tu celular.

Diseño y desarrollo de UI, UX y la App para iOS (iPhone/iPad) y Android, usando Cordova (PhoneGap), CoffeeScript, AngularJS, y AppGyver Steroids, un wrapper de Cordova para estas plataformas, con una UI diseñada enteramente en HTML5 + CSS, para que funcione como un híbrido entre tecnologías web y elementos nativos. Es la única app en colombia para pedir domicilios que soporta pagos en línea con tarjeta de crédito.

Se implementaron dos API para el backend de la aplicación. Uno principal en Ruby on Rails + MySQL, y un API de autenticación en Grape + DynamoDB. El deployment es realizado enteramente en instancias EC2 (Elastic Cloud Computing) de Amazon Web Services usando dokku, un PaaS (Platform as a Service) basado en Docker, y como servidor web de Rails se usa Puma.

Para las bases de datos se usa Amazon RDS (Relational Databases Service) y DynamoDB (una base de datos NoSQL).
La mayoría de recursos son servidos a través de Amazon S3 (Simple Storage Service)

Creación de un sistema de administración para los domicilios. Creación de una App para los domiciliarios usando las mismas herramientas que para la principal.
